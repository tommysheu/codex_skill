# Meeting Memo

## Instructions
1. Read the subtitle/transcript text.
2. Identify speakers, topics, decisions, action items, and unresolved issues.
3. Output Traditional Chinese meeting notes.
4. Keep uncertain speaker attributions marked clearly.
5. Preserve timestamps only when useful.
